## Description

Need to manage checks and cash payments?

This plugin generates bank slips for checks, cash or all other payment method.
Just select orders and export them in bank slip PDF file with the bulk action.
Join it with your bank deposit and you're done!

How to generate bank slip, step by step:
1. Select orders in the orders list
2. Select "Generate bank slip" in the bulk action
3. Fill the form: Date, Piece number and Account number
4. Click on "Generate bank slip"

The piece number et piece date will appear in the orders list, in the "Order status" column.

Moreover, orders with the same Bank Slip number will be grouped in accounting exports if you use Accounting for WooCommerce and the <a href="https://apps.avecnous.eu/produit/module-journal-de-banque-comptabilite-pour-woocommerce/?mtm_campaign=wp-plugin&mtm_kwd=accounting-for-woocommerce-bank-reconciliation&mtm_medium=wp-repo&mtm_source=premium">Bank Reconcilation addon</a>.

## Installation

1. Upload into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress admin

## Frequently asked questions

### Which payment gateways are supported?
Any. All orders can be exported. But we recommend to  group orders by payment method.

### Can I export bankslips data for accounting?
Not directly. But you can use Accounting for WooCommerce

https://wordpress.org/plugins/accounting-for-woocommerce/
